Getting Started with InfinityFit¶

Welcome to InfinityFit, a health management platform designed to help users achieve their fitness and wellness goals with ease and professionalism. This guide will help you set up and run the project.

Prerequisites¶

Before starting, ensure you have the following installed: - Python 3.8 or higher - pip (Python package manager) - Git (for cloning the repository)

Configuration¶

To configure InfinityFit, follow these steps:

1. Edit the configuration file: - Open the config.yaml file in the project directory. - Update the database URL, API keys, and other necessary settings.

2. Set environment variables (optional): - Create a .env file in the root directory. - Add the following variables:

` DATABASE_URL=your_database_url SECRET_KEY=your_secret_key `
